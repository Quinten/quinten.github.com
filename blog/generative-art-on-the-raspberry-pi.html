Generative art on the Raspberry Pi

19-03-2019

So the idea for this project is to have a very small computer unit, that automatically generates unique and random images and animations. By automatically i mean: the only technical requirement to start the flow of imagery is that you power on the computer and you connect it with a hdmi to a tv. A little bit like dvd player, only that it generates and computes the animations live, making images that are different each time.
